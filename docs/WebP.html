WebP

Description

WebP decoding and encoding functions.

The inputs of encoding functions and outputs of decoding functions are arrays of bytes representing pixels, with one byte per channel.

For example functions operating on arrays in RGB layout expect them to be in the following format:

import qualified Data.ByteString as Bytes
import Data.Word

Bytes.pack ([r, g, b, r, g, b, ..] :: Word8)

encodeRGB :: ByteString -> Width -> Height -> Quality -> ByteString Source

Encode an array of bytes in RGB layout at the specified width, height and quality. Quality ranges from 0 to 100. The value 0 corresponds to low quality and small output sizes, whereas 100 is the highest quality and largest output size.
